## Movies Reviews

Movie Reviews is a web app that enables to serach movies and rate them.

## Frameworks/Libraries used
    React
    Redux
    Thunk

## Preprcessor

## Steps to run the app 
    Run following commands to start the app
    1. npm install
    2. npm start
